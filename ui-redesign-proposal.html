UI Redesign · Proposal v1

极光控制台
Aurora Console

把管理后台从"暖色编辑部"重构为深空观测台：冷静深色基底承载海量指标，极光渐变（violet→cyan）作科技主调，一束暖琥珀信号色点亮关键操作与告警 —— 科技感与人文温度并存，彻底脱离 Anthropic 的米色衬线气质。

字体: Space Grotesk / Inter / JetBrains Mono 主色: 极光紫蓝渐变 信号色: 暖琥珀 半径: 10px 精准圆角 动效: 弹性 + 极光漂移
主推

A · 极光控制台

深色观测台，极光渐变 + 暖琥珀信号色。科技基底 + 人文高光，最契合你的偏好。

B · 青蓝实验室

单色青蓝体系，极简精准，偏"冷科技"。人文温度较弱，适合纯工程审美。

C · 暖阳熔炉

深色 + 饱和暖铜/熔金主色，温度优先。依然暗色，但更"热"，适合强品牌感。
